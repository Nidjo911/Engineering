# Django Project Template

A ready-to-use Django template designed for learning and quick project setup.

## Purpose

- **Learning Django concepts** - understand MVT architecture through a working example
- **Quick project starts** - skip repetitive setup and jump straight into development
- **Best practices reference** - see proper Django templating and structure in action

## What's Included

### Core Setup
- Static files configuration
- Templates directory structure
- Basic apps to get started
- Useful requirements.txt with essential packages

### Features
- **MVT Architecture** - Properly structured Model-View-Template pattern
- **Django Templating Engine** - Base template with template inheritance
- **CSS Integration** - Styling setup ready to customize
- **Dummy Data System** - Sample data in `data.py` (root folder) imported as module in `views.py`
- **Base Template** - Main HTML structure in `base.html`

## Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start development server
python manage.py runserver
```
